https://yuanzhizhu.github.io/js-template/

注：  
需要通过http(s)协议进行访问，不支持file://的方式。除非浏览器支持本地跨域。

语法1:  
```html
<template data-src="xxx.tpl"></template>
```

语法2:  
```html
<template data-render="{ name:'zzy', age:'20' }">
	<div>
		hello,my name is {{ name }},and my age is {{ age }}
	</div>
</template>
```
